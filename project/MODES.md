# Supported Modes

**Default Mode:** CHAT

| Mode | File | Description |
|------|------|-------------|
| 💬 CHAT | `.ai/agents/modes/chat/AGENT_CHAT.md` | Understanding, discussion, no code changes |
| 🔴 TDD | `.ai/agents/modes/tdd/AGENT_TDD.md` | Test-first, atomic commits, Red-Green-Refactor |
| ⚡ DEV | `.ai/agents/modes/dev/AGENT_DEV.md` | Speed, direct implementation, keep tests green |
| 🔍 REVIEW | `.ai/agents/modes/review/AGENT_REVIEW.md` | Structured branch diff review, analysis only, no code changes |
